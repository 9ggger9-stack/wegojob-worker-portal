# WeGoJob Portal v4

Czysty projekt React + Vite przygotowany do GitHub i Netlify.

Funkcje:
- wszystkie formularze onboardingowe,
- podpis na telefonie,
- zdjęcia paszportu, karty pobytu/dowodu i PESEL,
- integracja z Make,
- protokół odzieży,
- urlop,
- przycisk Wstecz,
- podgląd panelu koordynatora.

Uwaga: centralna lista pracowników w panelu koordynatora wymaga jeszcze scenariusza odczytu danych z Google Sheets. Obecna wersja panelu pokazuje dane zapisane na danym urządzeniu.
